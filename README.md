# ddl
<img src="docs/img/badges.svg">

Runtime DDL over [tinywasm/storage](https://github.com/tinywasm/storage):
CreateTable/DropTable/CreateDatabase/Sync/SyncSchema, plus the executable `ddl/conformance` contract for
SQL backends. A sibling of tinywasm/orm, not a dependency of it — see
[docs/ARCHITECTURE.md](docs/ARCHITECTURE.md) for the design.
